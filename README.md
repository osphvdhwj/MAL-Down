# MAL Down - Flutter Android App for MyAnimeList Image Downloading

This is the main README file for the MAL Down project hosted on GitHub.

- This project includes the complete Flutter app and Python scripts for MyAnimeList image management.
- For full documentation, see the DEVELOPMENT_GUIDE.md and GITHUB_SETUP_GUIDE.md files.

## Features
- MyAnimeList XML parsing
- Image download with retries
- EXIF metadata embedding compatible with AVES Gallery
- Intelligent organization by category and genre
- Material Design 3 UI optimized for Poco X5 Pro
