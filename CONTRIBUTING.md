# Contributing to MAL Down

Thank you for considering contributing to MAL Down!

## How to Contribute

1. Fork the repository
2. Create a new branch (`git checkout -b feature/AmazingFeature`)
3. Make your changes
4. Run tests (`flutter test`)
5. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
6. Push to the branch (`git push origin feature/AmazingFeature`)
7. Open a Pull Request

## Code Style

- Follow Dart style guide
- Run `flutter analyze` before committing
- Run `flutter format .` to format code
- Write tests for new features

## Pull Request Process

1. Update README.md with details of changes
2. Update DEVELOPMENT_GUIDE.md if needed
3. Ensure all tests pass
4. Request review from maintainers

## Code of Conduct

Be respectful and inclusive. Follow GitHub Community Guidelines.
