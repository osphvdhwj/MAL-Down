# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release
- MyAnimeList XML import functionality
- Image download with 5 retry attempts
- EXIF metadata embedding (AVES compatible)
- Intelligent folder organization (Anime/Manga/Hentai)
- Genre-based subfolder creation
- Placeholder image generation
- Background download with WorkManager
- Progress tracking and notifications
- Material Design 3 UI
- Dark mode support
- Search and filter functionality
- Gallery integration
- Offline-first architecture

### Features
- Clean Architecture + MVVM pattern
- Riverpod state management
- Sqflite local database
- Dio HTTP client
- Complete test coverage

### Optimizations
- Optimized for Poco X5 Pro (120Hz AMOLED)
- Efficient image caching
- Battery-conscious background processing
